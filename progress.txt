## Codebase Patterns
- LibriVox audio fixtures: download from archive.org with Range header (64kbps → 8KB/s, so 200KB ≈ 25s)
- setup.py is a Python script (not shell), checks ffmpeg via shutil.which + ollama via HTTP to localhost:11434
- `requests` is pre-installed in codespace; `pydub`, `faster-whisper`, `pydantic`, `pytest` are NOT — install via pip
- Audio fixture is at tests/fixtures/sample_chapter.mp3 (LibriVox Art of War ch01, ~25s, contains Disclaimer)
- repository.py: top-level key is str(librivox_project_id); upsert accepts {book_metadata: {...}, chapter: {...}}
- Atomic write pattern: tempfile.mkstemp in same dir → write → os.replace; cleanup tmp on failure
- boundary_detector.py: inject a fake requests.Session for testing — session.post.side_effect = [mock_response, ...]; Ollama response shape is {"message": {"content": "<json string>"}}
- BoundaryResult = Union[NoDisclaimer, AnchorWord] — use isinstance() to branch on the result type
- audio_fetcher.py: use @staticmethod + @contextmanager on AudioFetcher.fetch(); inject session for testing; mock session.get.side_effect = [exc, ok_response] for retry tests; mock response needs raise_for_status() and iter_content()
- analyzer.py: inject fake detector via `detector` param (default: detect_boundary); faster-whisper word.end is np.float64 — call float() on it; abs(delta) > N also yields np.bool_ — call bool() to get Python bool
- ffmpeg not in Codespace: install static-ffmpeg via pip, call static_ffmpeg.add_paths() in tests/conftest.py (pydub calls ffmpeg/ffprobe as subprocess, so PATH must be set before any pydub import)
- faster-whisper strips punctuation inconsistently — normalize both anchor word and token map words with re.sub(r"[^a-z0-9]+", "", w.lower()) when matching
